Setup sekali saja

Supaya tombol Generate bisa langsung trigger workflow tanpa buka GitHub, perlu Personal Access Token (PAT) dari akun GitHub Anda. PAT disimpan hanya di HP ini (localStorage).

  1. Tap tombol di bawah → halaman GitHub akan terbuka
  2. Login kalau diminta
  3. Scope workflow sudah otomatis tercentang
  4. Set Expiration "No expiration" atau 1 tahun
  5. Klik Generate token di bawah halaman
  6. Copy token yang muncul (mulai dengan ghp_…)
  7. Balik ke app, paste di kolom di bawah
